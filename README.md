# Shorty
 A re-programable 3x3 board with Mechanical Switches using Raspberry Pi PICO

## Re-Programing Guide
- for <b>C/C++</b> users use this guide [here](https://www.tomshardware.com/how-to/program-raspberry-pi-pico-with-arduino-ide) using arduino IDE
- for users how arae more familiar with python there is a easy way around it:
    - you can use <b>micro-python</b> using this guide [here](https://www.tomshardware.com/how-to/raspberry-pi-pico-setup)
    - or you can use <b>circuit-python</b> by adafruit guide [here](https://learn.adafruit.com/welcome-to-circuitpython/installing-circuitpython)
